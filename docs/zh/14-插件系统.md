# 13 - 插件系统

本章你将学会：把“工具 + 钩子 + 权限规则”打包成一个插件，一次编写，多处复用。

## 1. 插件是什么

当你写了一套相关的能力（比如一个“时区插件”：一个时间工具 + 一个审计钩子 + 一条权限规则），
可以打包成 `Plugin`，然后 `agent.use(plugin)` 一行装上。

## 2. 用 PluginBuilder 构建

```ts
import { PluginBuilder, BaseTool, type ToolContext } from 'mochikit';

// 一个工具
class CurrentTimeTool extends BaseTool {
  readonly definition = {
    name: 'current_time',
    description: '获取当前 ISO 时间。',
    input_schema: { type: 'object', properties: {} },
  };
  async execute(): Promise<string> {
    return new Date().toISOString();
  }
}

// 打包成插件
const timePlugin = new PluginBuilder('time-plugin')
  .tool(new CurrentTimeTool())                     // 加工具
  .hook('PostToolUse', (p) => {                    // 加钩子
    const payload = p as { tool: { name: string } };
    console.log('[audit]', payload.tool.name);
  })
  .rule({                                          // 加权限规则
    name: 'allow-time',
    tools: ['current_time'],
    check: () => 'allow' as const,
  })
  .build();
```

## 3. 装到 Agent

```ts
import { Agent, AnthropicAdapter, loadConfig, AllowAllResolver, PermissionManager } from 'mochikit';

const cfg = loadConfig();
const agent = new Agent({
  name: 'plugin-demo',
  llm: new AnthropicAdapter({ apiKey: cfg.apiKey, baseURL: cfg.baseUrl }),
  model: cfg.model,
  systemPrompt: '你可以用 current_time 工具查时间。',
  permission: new PermissionManager({ resolver: new AllowAllResolver() }),
});

agent.use(timePlugin); // 一行装载

console.log(await agent.run('现在几点？用工具查。'));
```

`Agent` 实现了 `PluginHost` 接口，所以 `use(plugin)` 会把插件里的工具/钩子/规则全注册进来。

## 4. 手写 Plugin 接口

不用 Builder，直接实现接口也行：

```ts
import type { Plugin, PluginHost } from 'mochikit';

const myPlugin: Plugin = {
  name: 'my-plugin',
  install(host: PluginHost) {
    host.registerTool(new CurrentTimeTool());
    host.registerHook('PostToolUse', (p) => { /* ... */ });
    host.registerPermissionRule({ /* ... */ });
  },
};

agent.use(myPlugin);
```

## 5. 跨 Agent 共享插件

用 `PluginRegistry` 收集一组注册，再批量应用到多个 Agent：

```ts
import { PluginRegistry } from 'mochikit';

const registry = new PluginRegistry();
registry.install(timePlugin);
registry.install(anotherPlugin);

// 应用到 agentA 和 agentB
registry.applyTo(agentA);
registry.applyTo(agentB);
```

## 6. PluginHost 能做什么

`PluginHost`（Agent 实现了它）提供三个方法：

- `registerTool(tool)`：注册工具
- `registerHook(event, cb, priority?)`：注册钩子
- `registerPermissionRule(rule)`：注册权限规则

插件 `install(host)` 时拿到 host，调用这些方法完成注册。
