# 17 - 完整实战示例

本章你将学会：把前面所有知识串起来，做一个端到端的实用示例。

## 场景

我们要做一个“技术调研助手”：用户给一个主题，系统自动：
1. 用 Manager-Worker 拆任务 → 一个 Worker 搜资料，另一个算相关数据
2. 记忆系统记录用户的偏好
3. 最后链式润色产出报告

## 完整代码

```ts
import {
  Agent,
  AnthropicAdapter,
  loadConfig,
  ManagerWorker,
  SequentialChain,
  MarkdownMemory,
  InMemoryTaskStore,
  InMemoryMessageBus,
  createBashTool,
  createWebSearchTool,
  createWebReaderTool,
  createMemoryTools,
  createTaskTools,
  AllowAllResolver,
  PermissionManager,
  PluginBuilder,
  BaseTool,
  type ToolContext,
} from 'mochikit';

// =========== 1. 配置 ===========
const cfg = loadConfig();
const llm = () => new AnthropicAdapter({ apiKey: cfg.apiKey, baseURL: cfg.baseUrl });
const perm = () => new PermissionManager({
  resolver: new AllowAllResolver(),
  rules: [{
    name: 'no-rm-rf',
    tools: ['bash'],
    check: (ctx) => (/rm\s+-rf/.test(String(ctx.tool.input.command)) ? 'ask' : 'passthrough'),
    reason: 'dangerous command',
  }],
});

// =========== 2. 记忆系统 ===========
const memory = new MarkdownMemory({ dir: './.mochikit/examples/demo' });

// =========== 3. 自定义工具：统计文章字数 ===========
class WordCounterTool extends BaseTool {
  readonly definition = {
    name: 'word_count',
    description: '给定一段文本，统计字数。',
    input_schema: {
      type: 'object',
      properties: { text: { type: 'string' } },
      required: ['text'],
    },
  };
  async execute(input: Record<string, unknown>): Promise<string> {
    const text = this.requireString(input, 'text');
    const count = text.replace(/\s+/g, '').length;
    return `总字数：${count}`;
  }
}

// =========== 4. 插件：调研工具包 ===========
const researchPlugin = new PluginBuilder('research-kit')
  .tool(new WordCounterTool())
  .tool(createWebSearchTool(cfg.webApiKey))
  .tool(createWebReaderTool(cfg.webApiKey))
  .hook('PostToolUse', (p) => {
    const payload = p as { tool: { name: string } };
    console.log(`[research] 工具 ${payload.tool.name} 执行完毕`);
  })
  .build();

// =========== 5. Worker Agent ===========
const researcherWorker = new Agent({
  name: 'researcher',
  llm: llm(), model: cfg.model,
  systemPrompt: '你是调研员。用 web_search 搜索内容，必要时用 web_reader 阅读页面，用 word_count 统计字数。简洁汇报关键发现与字数。',
  permission: perm(), maxTurns: 6,
});
researcherWorker.use(researchPlugin);

const calculatorWorker = new Agent({
  name: 'calculator',
  llm: llm(), model: cfg.model,
  systemPrompt: '你是计算员。用 bash 工具做计算。只返回计算结果。',
  tools: [createBashTool()],
  permission: perm(), maxTurns: 4,
});

// =========== 6. Manager Agent ===========
const manager = new Agent({
  name: 'pm',
  llm: llm(), model: cfg.model,
  systemPrompt:
    '你是项目经理。把调研任务拆解：让 researcher 搜索并统计，让 calculator 做数据计算。用 spawn_teammate 委派，然后汇总成一份带项目信息的报告。用 memory_write 保存用户偏好。',
  permission: perm(), memory,
  tools: createMemoryTools(memory),
  maxTurns: 8,
});

const mw = new ManagerWorker({
  manager,
  workers: [
    { name: 'researcher', agent: researcherWorker },
    { name: 'calculator', agent: calculatorWorker },
  ],
});

// =========== 7. 报告润色 Agent ===========
const polisher = new Agent({
  name: 'polisher',
  llm: llm(), model: cfg.model,
  systemPrompt: '把输入文本润色成一份结构清晰的技术调研报告，加标题和小标题。',
  permission: perm(), maxTurns: 3,
});

// =========== 8. 流水线：Manager-Worker → 润色 ===========
async function main() {
  // 先记偏好
  await memory.add({
    name: '用户偏好格式',
    type: 'feedback',
    description: '技术调研报告格式偏好',
    body: '报告宜结构清晰，标题 H1，小节 H2，每节含要点列表。',
  });

  // Manager-Worker 调研
  console.log('=== 启动 Manager-Worker 调研 ===');
  const draft = await mw.run('调研 "TypeScript 装饰器" 并统计相关数据。');

  // 润色链
  console.log('=== 润色 ===');
  const chain = new SequentialChain({ agents: [polisher], sharedMemory: memory });
  const final = await chain.run(draft);

  console.log('\n========== 最终报告 ==========\n');
  console.log(final);

  // 查看留下的记忆
  const all = await memory.list();
  console.log(`\n共 ${all.length} 条记忆`);
}

main().catch((e) => { console.error(e); process.exit(1); });
```

## 这个示例用到了什么

| 功能 | 所在位置 |
|---|---|
| `Agent` + `AnthropicAdapter` | 基础运行 |
| `createBashTool` / `createWebSearchTool` / `createWebReaderTool` | 内置工具 |
| `BaseTool`（`WordCounterTool`） | 自定义工具 |
| `PluginBuilder` / `agent.use()` | 插件封装 |
| `MarkdownMemory` + `createMemoryTools` | 记忆系统 |
| `PermissionManager` + 规则 | 权限控制 |
| `ManagerWorker` + `spawn_teammate` | 多智能体委派 |
| `SequentialChain` | 顺序链润色 |
| `Hook`（`PostToolUse`） | 审计钩子 |
